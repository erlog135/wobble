# Wobble
A soft body physics watchface for Pebble, inspired by the JellyCar game series. This app was created within the Cursor editor.

**Features**
- Semi-responsive layout
- Big colorful jelly numbers that wobble when the time changes, or when you shake your watch. These are soft body physics objects whose logic resides in the `src/physics` folder.
- A battery power bar. Its width scales proportionally to the watch's battery level.
- A date & day of week indicator.
